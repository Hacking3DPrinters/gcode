# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1     | :x:                |
| 0.1b    | :x:                |
| 0.1a    | :x:                |

## Reporting a Vulnerability

If the vulnerability is low, please open a request on our issues page.

If the vulnerability is high/critical, please email Hacking3DPrinters.
